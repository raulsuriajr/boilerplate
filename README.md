### Installation
* `./setup.sh` to run setup.
* Install like normal

### Features
Drupal version 9

Compatible with composer 2 only

`./vendor/bind/drush` to use drush

### Modules
* Back up and migrate
* Adminimal Admin Toolbar
* Pathauto
* Twig Tweak
* Twig Field Value
* Editor Advanced Link
* Image effects

To enable these modules run `./setup.sh enmod`
